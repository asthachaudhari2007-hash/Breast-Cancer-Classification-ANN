# Breast Cancer Classification using Artificial Neural Networks

## Overview
This project uses Artificial Neural Networks (ANN) to classify breast tumors as benign or malignant using the Wisconsin Breast Cancer Dataset.

## Technologies Used
- Python
- TensorFlow
- Keras
- Scikit-learn
- Pandas
- NumPy
- Plotly
- Seaborn
- Matplotlib

## Features
- Data preprocessing and feature scaling
- ANN model with hidden layers
- Binary classification
- Model evaluation
- Confusion Matrix visualization
- Training accuracy graph

## Model Performance
- Training Accuracy: 99.5%
- Validation Accuracy: 97.8%
